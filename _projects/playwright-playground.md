A sandbox repository for exploring [Playwright](https://playwright.dev/) capabilities — from basic API usage to advanced patterns.

Used as a personal learning space and reference when evaluating Playwright features for production use at Allegro.

**Topics covered:**
- Browser contexts and parallel execution
- API testing with Playwright's request context
- Visual comparisons
- Mobile emulation
- Authentication state management
