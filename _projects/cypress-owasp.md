An end-to-end test automation suite targeting the [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/), built with **Cypress** and **JavaScript**.

Covers core user flows including registration, login, shopping cart, and checkout, as well as selected security-related scenarios.

**Key highlights:**
- Custom Cypress commands for reusable actions
- Network interception with `cy.intercept()`
- Fixtures for test data management
- Mochawesome HTML reports
