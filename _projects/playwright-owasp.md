A comprehensive end-to-end test automation suite targeting the [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) — a deliberately insecure web application used for security training.

Built with **Playwright** and **TypeScript**, the suite covers key user flows and verifies both functional correctness and security-related behaviours.

**Key highlights:**
- Page Object Model architecture
- Parallel test execution
- HTML test reports via Playwright's built-in reporter
- CI-ready configuration
